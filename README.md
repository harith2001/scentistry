# Scentistry

Premium candles e-commerce built with Next.js + Firebase.

## Quickstart

1. Create a Firebase project and enable:
   - Authentication (Email/Password)
   - Firestore Database
   - Storage
2. Copy `.env.example` to `.env.local` and fill values.
3. Install and run:

```powershell
npm install
npm run dev
```

Open http://localhost:3000.

## Environment variables
See `.env.example`.

## Deploy
Recommended: Vercel for web, Firebase for DB/Storage/Functions.
